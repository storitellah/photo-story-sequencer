# Lightbox Lab — Photo Story Sequencer

A single-page, local-first web app for building photo stories: rate photos on a contact sheet, sequence them into a story timeline, play with pairing suggestions, and preview the edit as an Instagram carousel, zine or exhibition wall.

**Everything stays on your computer.** No backend, no login, no uploads, no external libraries. Original image files are never modified, resized or renamed.

## Use it

Open `index.html` in any modern browser (works fully offline), then drag a folder of photos onto the page.

Keyboard shortcuts: `1` Keep · `2` Maybe · `3` Reject · `H` Hero · `O` Opening · `C` Closing · `←`/`→` navigate · `Esc` close panel.

Project data (ratings, captions, sequence) is saved in your browser's local storage and can be exported/imported as a JSON backup from the Export menu.
